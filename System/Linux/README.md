## ★常用命令行

### ◇Unix/Linux命令

| command       | is                                         | show                                            |
| ------------- | ------------------------------------------ | ----------------------------------------------- |
| `ls -F`       | 简单显示当前目录下的文件和目录             | `node/`                                         |
| `ls -ah`      | 显示所有的目录和文件，包括 `.`开头的       | `node/ .pip`                                    |
| `ls -lh`      | 详细地显示文件的各种信息，如大小、日期时间 | `drwxr-xr-x 2 root root 4.0K Dec  4 17:42 node` |
| `mv -i f1 f2` | 改名字                                     | `nodejs/`                                       |
|               |                                            |                                                 |
|               |                                            |                                                 |
